## AES--128-File-encryption-system
**AES-128 File Encryption System** is a security technique that encrypts files using a 128-bit secret key, making the data unreadable to unauthorized users. It provides fast and reliable protection for sensitive information during storage and transmission.

## Features

* Encrypts files using AES-128 algorithm
* Secure password-based key generation
* Decrypts encrypted files with the correct key
* Protects sensitive data from unauthorized access
* Simple and user-friendly interface

## Technologies Used

* Python
* PyCryptodome
* AES-128 Encryption

## Note

This project demonstrates the implementation of AES-128 symmetric encryption for securing digital files. Done with a bit of AI assistance.
